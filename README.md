# Flixx App
In this project, I build a movie info application that fetches data from [TMDB](https://www.themoviedb.org/) API and display it in a nice User Interface. This app has the following features:

- Displays the most popular movies
- Displays the most popular TV shows
- Shows specific movie details
- Shows specific TV show details
- Displays the 'Now Playing' movies in a slider
- Allows users to search for movies and TV shows
- Has pagination for search results
